Documentación Pública - API de Productos

Todos los endpoints disponibles son de solo lectura (GET).

GET /api/public

Devuelve todos los productos.

Ejemplo de respuesta:

[
  {
    "_id": "abc123",
    "name": "Producto 1",
    "description": "Descripción del producto",
    "price": 123.45,
    "image": "https://ejemplo.com/imagen.jpg"
  }
]

GET /api/public/:id

Devuelve un producto específico por su ID.

Ejemplo de respuesta:

{
  "_id": "abc123",
  "name": "Producto 1",
  "description": "Descripción del producto",
  "price": 123.45,
  "image": "https://ejemplo.com/imagen.jpg"
}
